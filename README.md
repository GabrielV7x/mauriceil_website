# MauriCeil Interiors Website

A single-page responsive website for **MauriCeil Interiors**, showcasing their services, gallery, about section, and contact information.
Built with **React** and **SCSS**, designed mobile-first and fully responsive for tablet and desktop screens.

---

## Features

- Mobile-first responsive design
- Hero section with bottom-left overlay
- Sticky navbar with burger menu on mobile
- Smooth scrolling to sections
- Sections included: Hero, About, Gallery, Services, Contact, Footer
- Contact button linked to WhatsApp for instant inquiries
- Fully styled with SCSS variables and **CSS clamp()** for responsive scaling
- Modern typography using Google Fonts

---

## Technologies Used

- React (Vite)
- SCSS / SASS
- CSS clamp() for responsive typography and spacing
- Netlify (for hosting)
- Google Fonts
